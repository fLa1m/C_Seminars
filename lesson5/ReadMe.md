## Задача №1. Возвести в степень

Составить функцию, возведение числа N в степень P. int power(n, p) и привести пример
ее использования.

Данные на входе: Два целых числа: N по модулю не превосходящих 1000 и P > 0  
Данные на выходе: Одно целое число

**_Пример №1_**  
Данные на входе: 8 3  
Данные на выходе: 512

**_Пример №2_**  
Данные на входе: -5 2  
Данные на выходе: 25
